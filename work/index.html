  1. 'L' Bus

    iOS CTA Tracker

    'L' Bus is a Chicago Transit Authority (CTA) Bus and Train tracker.

    In the app, the user is able to select a route and stop in order to view approximate vehicle arrival times at the selected stop.

    It features the ability to favorite stops for easier access.

    Users are also able to follow a specific vehicle to view where it currently is.

    Lastly, 'L' Bus is able to read aloud the upcoming arrival times, refreshing every 45 seconds to provide updated arrival times.
